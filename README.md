# 💱 Conversor de Moedas 
Um conversor de moedas moderno e responsivo.Desenvolvido com HTML, CSS e JavaScript vanilla, sem dependências externas.

## ✨ Funcionalidades

- **Conversão de várias moedas**: Converta entre 6 moedas diferentes (Real, Dólar, Euro, Libra, Bitcoin e Rublo Russo)

- **Interface Responsiva**: Design adaptado para desktop, tablet e mobile


## 🛠️ Tecnologias Utilizadas

- **Frontend**: HTML5, CSS3, JavaScript ES6+
- Navegador moderno com suporte a ES6+ (Chrome, Firefox, Safari, Edge)
- Nenhuma instalação adicional necessária

## 🚀 Como Usar

### . **Utilize o conversor**
   - Selecione a moeda de **origem** (Converter de)
   - Selecione a moeda de **destino** (Converter para)
   - Digite o valor
   - Clique em **Converter**
   


4. **Conversão**
   - Valor de origem × taxa de origem = valor em EUR
   - Valor em EUR ÷ taxa de destino = valor final convertido
